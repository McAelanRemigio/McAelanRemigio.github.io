What this is

SDSU_Undergraduate_Career is not a single project—it’s a living portfolio that documents applied analytics work developed alongside my undergraduate studies. It captures projects, notes, and workflows built in response to real problems, not course prompts.

How it’s used

  • Documenting applied analytics projects that don’t warrant full standalone case studies.
  • Tracking learning through reproducible notebooks, notes, and experiments.
  • Maintaining a transparent record of how ideas evolve into usable workflows.
  • Supporting other projects (e.g., Career Services analytics, Rubin workflow) with shared methods.

Scope

The repository spans multiple semesters and includes work across data analysis, reporting, exploratory research, and workflow design—built iteratively as skills and context evolve.

Focus areas

  • Applied analytics
  • Reproducible workflows
  • Documentation & learning in public
  • Translating ambiguity into structure

Tools

Python · R · SQL · Jupyter · Data visualization · Documentation

Why this matters

Strategy and analytics work rarely emerges fully formed. This repository demonstrates how I approach ambiguous problems: explore, document, refine, and reuse. It emphasizes process, clarity, and iteration—skills that carry across analytics, operations, and decision-making roles.
